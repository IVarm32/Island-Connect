## 2024-05-24 - [Standard Cursor Affordance]
**Learning:** Overriding the default browser cursor (e.g., using `cursor: default` on links) is a significant UX anti-pattern. It breaks the user's mental model of what is interactive and reduces the perceived responsiveness of the interface. Standard affordances like the pointer cursor are critical for intuitive navigation.
**Action:** Avoid hardcoding `cursor: default` on interactive elements. Always ensure that clickable elements provide clear visual feedback, including the standard pointer cursor.
